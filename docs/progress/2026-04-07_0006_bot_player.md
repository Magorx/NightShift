## Work Done

Implemented all BOT.1–BOT.6 cards: an autonomous AI that plays the game by itself.

### Architecture

**`tests/bot/bot_player.gd`** — base class (extends SimulationBase):
- `BotBrain` inner class: grid-based decision-making AI
  - One decision per `ticks_per_decision` frames (default 60 = 1/sec)
  - Moves one grid cell per decision toward current target (cardinal, skip walls)
  - Teleports player to current grid position each step
  - Three strategies: RANDOM, GREEDY, LINE_BUILDER
- `BotMetrics` inner class: tracks buildings by type + items delivered per run
- Clears world-generated deposits before each run (controlled deposit set of 9)
- Timeout set to 180s (builds can be heavy)

### Strategies
- **RANDOM** — random walk + weighted placement (conveyor 50%, smelter 15%, splitter 10%, source/sink 5% each). 30% conveyor chain extension (1–4 extra)
- **GREEDY** — walks to nearest undrilled deposit, drills it, routes 4 conveyors → sink. Systematic: drills all 9 deposits
- **LINE_BUILDER** — finds deposit with clear axis (≥5 free cells), builds full drill→conveyor×5→sink line. Most efficient: delivered 266 items in 60s standalone run

### Runnable Sims
- `sim_bot_random.gd` — 60s random (also: `--visual` to watch)
- `sim_bot_greedy.gd` — 60s greedy
- `sim_bot_linebuilder.gd` — 60s line builder (also: `--visual`)
- `sim_bot_batch.gd` — 3×30s runs back-to-back with comparison table
- `sim_bot_smoke.gd` — 30s smoke test, asserts ≥3 buildings placed
